Welcome to Brain Break

Brain Break is a game based on "Simon Says" game, the purpose of this code is learn React.js. Version 1.0 include 4 different levels, variations of the original game. Code and game experience will be improved in future versions.

Libraries

Underscore 1.8, React 0.3.13, Jquery 1.11, in the future gulp and nodeJs will be implemented.
